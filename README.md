# BMI Calculator

## Description

BMI Calculator is a simple Python application that calculates a person's Body Mass Index (BMI) using their weight and height. The program takes user input, performs the BMI calculation, and categorizes the result according to standard BMI ranges.

## Features

* Calculate Body Mass Index (BMI)
* User-friendly command-line interface
* Displays BMI category
* Input validation and error handling
* Simple and easy-to-understand Python code

## Technologies Used

* Python 3

## How to Run

1. Open Command Prompt, PowerShell, or VS Code Terminal.
2. Navigate to the project folder.
3. Run the following command:

```bash
py bmi_calculator.py
```

## Example Output

```text
Enter your weight (kg): 70
Enter your height (m): 1.75

Your BMI is: 22.86
Category: Normal Weight
```

## BMI Categories

| BMI Range      | Category      |
| -------------- | ------------- |
| Less than 18.5 | Underweight   |
| 18.5 - 24.9    | Normal Weight |
| 25.0 - 29.9    | Overweight    |
| 30.0 and above | Obese         |
